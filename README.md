🧾 Sistema de Facturación para Restaurante (Python + Tkinter)

Aplicación de escritorio desarrollada en Python que simula un sistema de facturación para un restaurante.
Permite seleccionar productos, calcular costos automáticamente, generar recibos y guardarlos como archivo.

🚀 Funcionalidades

✅ Selección de productos (comida, bebidas y postres)
✅ Ingreso de cantidades dinámico
![App](imagenes/Sistema-de-Facturaci-n-para-Restaurante-Python-Tkinter-
.png)
✅ Cálculo automático de:

Subtotal
Impuestos (7%)
Total

✅ Generación de recibo detallado
✅ Guardado del recibo en archivo .txt
✅ Botón de reinicio del sistema
✅ Calculadora integrada 🧮

🖥️ Interfaz

El sistema cuenta con una interfaz gráfica construida con Tkinter, dividida en:

Panel de productos 🍔
Panel de costos 💰
Área de recibo 🧾
Calculadora integrada 🧮
🧠 Tecnologías utilizadas
Python 🐍
Tkinter (GUI)
datetime
random
filedialog / messagebox
⚙️ Instalación y ejecución
Clona el repositorio:
git clone https://github.com/TU-USUARIO/TU-REPO.git
cd TU-REPO
Ejecuta el proyecto:
python nombre_del_archivo.py

⚠️ No requiere instalación de librerías externas (solo Python estándar)

📋 Cómo usar
Selecciona los productos marcando los checkboxes
Ingresa las cantidades
Presiona "Total" para calcular
Presiona "Recibo" para generar el detalle
Presiona "Guardar" para exportar el recibo
Usa "Resetear" para comenzar nuevamente
